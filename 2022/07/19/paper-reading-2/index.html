论文阅读-A review on absolute visual localization for UAV

前情：GNSS-INS方案存在问题

  1. GNSS通过无线信号测量，信号可被环境遮挡、反射、干扰，导致较大误差。
  2. 存在安全漏洞，攻击者可通过信号篡夺得到无人机控制权。

故提出基于视觉和视觉辅助的替代或补充方案。

RVL(Relative visual localization)

即帧到帧定位，包括视觉里程计(VO)、SLAM。

  • VO、VIO：光流分析，匹配当前帧与先前帧差异估计位姿。
  • SLAM：较VO，增加闭环检测和BA，若无闭环情况效果等同于VO。

核心问题：误差积累即随时间漂移，虽能融合INS惯性数据，但仍没解决误差积累。

AVL(Absolute visual localization)

即根据无人机当前的观测，在地图中进行匹配，从而在地图中对自身位置进行定位。地图可是卫星地图或自行采集的地图。

优点：不受时间漂移的影响。

难点：

  1. 地图集由不同传感器在不同时间、照明水平、分辨率拍摄带来的匹配挑战。
  2. 减少在全局地图匹配时的搜索空间。

图像匹配：

1. Template matching

直接匹配或密集匹配，将无人机当前视图作模板在地图中进行匹配。

缺点：相似性判断的计算成本高。

  • Van Dalen G.J., Magree D.P., Johnson E.N.《Absolute localization using image alignment and particle filtering》：

    normalized cross-correlation(NCC)，作为概率密度函数在粒子滤波中，执行无人机视图和全局参考地图的密集图像对齐。

    用高度信息在模板匹配前对视图缩放。

  • Yol A., Delabarre B., Dame A., Dartois J.-E., Marchand E.《Vision-based absolute localization for unmanned aerial vehicles》

    mutual information (MI)作为相似性度量使用，测两个图像的共享信息量。

  • Wan X., Liu J., Yan H., Morgan G.L.K.《Illumination-invariant image matching for autonomous UAV localisation based on optical sensing》

    基于光度不变的相位相关（illumination invariant Phase Correlation）的定位方案。

    Phase Correlation (PC)，一种基于傅里叶变换特性的模板匹配或图像对齐方法。算法利用两幅相似图像间的变化会在频域产生线性相位差，对太阳位置引起的光照变化不敏感。基于此，作者提出了一个基于卫星图像模板匹配的AVL方案。

  • Patel B.《Visual Localization for UAVs in Outdoor GPS-denied Environments》

    从MI获得normalized information distanc(NID)，但不依赖图像间的重叠量，因此定位更具鲁棒性。该方法结果与网格搜索的结果相似，但计算量要少。

    限制是假设UAV frame和参考图像间的旋转变化为0.

2.Feature points matching

模板匹配的有效替代方案，涉及特征点检测（角点、显著点检测）和描述子提取(gradient histograms from SIFT and the binary tests from BRIEF)。

本节中大多使用特征点匹配和统计滤波的组合实现定位。

  • Seema B.S., Hemanth K., Naidu V.P.S.《Geo-registration of aerial images using RANSAC algorithm》 && Saranya K.C., Naidu V.P.S., Singhal V., Tanuja B.M.《Application of vision based techniques for UAV position estimation》

    基于NCC的模板匹配方法 与 基于RANSAC的SURF特征匹配方法对比。

    NCC模板匹配方法先对图像和地图做边缘检测，然后以UAV图像为模板进行滑动窗口匹配，最大响应即无人机位置。特征点方法，提取无人机图像和地图的SURF特征匹配，用RANSAC除去异常值，计算剩余特征点匹配的几何变换恢复地图中无人机图像位置。

    NCC执行速度比RANDAC快，但NCC对尺度变化、旋转、噪声和模糊敏感，因此更适合旋转和尺度已知的情况。RANDAC鲁棒性更强，但最少特征数量的需求以及特征检测和提取时间不是恒定的事实。

  • Shan M., Wang F., Lin F., Gao Z., Tang Y.Z., Chen B.M.《Google map aided visual navigation for UAVs in GPS-denied environment》

    开发了一个框架，结合定向梯度直方图 (HOG) 、粒子滤波和光流(OF)，用HOG矢量距离估计位置，未通过阈值测试，再将OF用于估计。

  • Chiu H.P., Das A., Miller P., Samarasekera S., Kumar R.《Precise vision-aided aerial navigation》

    IMU与geo-referenced imagery提供的定位融合。固定滑动窗口估计飞机，两种估计过程：一是2D-3D tie-points and geo-registered feature tracks，二是逐帧跟踪特征扩展过程一来传递3D绝对信息。

  • Mantelli M., Pittol D., Neuland R., Ribacki A., Maffei R., Jorge V., Prestes E., Kolberg M.《A novel measurement model based on abBRIEF for global localization of a UAV over satellite images》

    基于Brief提出abBrief描述子，使用Lab颜色表示的a,b通道非灰度图，用 quantization levels减少噪声非高斯模糊，在整个图像上随机选择固定数量的像素对非在特征点周围的区域中选择了像素对（跳过关键点检测减少计算时间），为每个图像生成一个全局描述符。

    粒子滤波器(MCL)，用于估计无人机在状态空间中的位置，距离模型由粒子描述符和当前UAV图像描述符之间的汉明距离组成，运动更新基于VO系统（非重点）。

  • Masselli A., Hanten R., Zell A.《Localization of unmanned aerial vehicles using terrain classification from aerial images》

    基于地形分类的粒子滤波方法，地形为：草地，灌木丛，道路，建筑物。在图像网格的每个单元提取ORB描述子依据随机森林模型分类，地图和UAV图像进行地形分类，然后用粒子滤波的方法定位，仅依赖视觉。

  • Shan M., Charan A.《Google map referenced UAV navigation via simultaneous feature detection and description》

    同时使用特征检测和提取将UAV图像和地图匹配，用maximal self-dissimilarities (MSD)检测特征点，local self-similarities (LSS)提取描述子，OF缩小地图的搜索空间，计算匹配特征点描述子的欧几里得距离之和，与OF预测过大的估计位姿被排除。

    飞行路径结果相对接近GPS定位，远优与单独OF。

  • Couturier A., Akhloufi M.A.《Relative visual localization (RVL) for UAV navigation》

    基于AVL的RVL框架。具体如下图：(ORB效果最佳)

    在《UAV navigation in GPS-denied environment using particle filtered RVL》中，引入带IMU运动模型的粒子滤波器和使用vote-based feature point matching测量模型，减少数据库搜索空间、计算时间和误报率，消除了计算时间对环境大小的依赖，实现了有限的执行时间。

    在 《Conditional probabilistic relative visual localization for unmanned aerial vehicles》提出粒子滤波的条件判断，用于粒子滤波器的更新步骤，考虑特征点的匹配质量，通过消除大多数剩余的误匹配特征点，实现更快更准确稳定的求定位解。该框架下，SURF效果较佳。

3. Deep learning

基于深度学习的定位还在起步阶段，主要原因是难以构建本地化的端到端架构。RVL需要参考图，且查询地图是动态变化的，CNN的拟合速度不能满足要求；AVL，参考地图太大，现有GPU不能满足要求。此外，缺乏相关公开的数据集。

  • Amer K., Samy M., ElHakim R., Shaker M., ElHelw M.《Convolutional neural network-based deep urban signatures with application to drone localization》

    提出deep urban signatures概念，用CNN根据视觉外观计算不同城市区域的独特特征。目的是通过分类方法探索不同城市地区或地区结构和组织间的差异（城市发展规划（或缺乏）、发展速度、建筑材料等因地区而异）。

    采用分层定位机制：district-level localization ， neighborhood-level localization.

    • district-level localization：训练网络对七个地区之一的查询图像分类，以减少搜索空间。
    • neighborhood-level localization：使用来自同一CNN的特征图执行最邻近搜索，从14th卷积层提取查询图像和参考图的特征图并比较，最接近查询图像的参考图的位置用于位置估计。

    主要缺点：在使用重定位之前需要在特定区域的数据集上训练网络。

  • Nassar A., Amer K., ElHakim R., ElHelw M.《A deep cnn-based framework for enhanced aerial imagery registration with applications to uav geolocalization》&& Nassar A., ElHelw M.《Aerial imagery registration using deep learning for uav geolocalization》

    将传统计算机视觉(SIFT+ORB)和CNN结合，基于卫星地图。

    UAV的位姿追踪：每3帧进行一次UAV到地图的校准(SIFT特征+RANSAC)，每帧计算前后序列帧间的里程计(ORB+RANSAC)

    基于语义分割的后端优化：将查询图像和匹配到的地图区域用U-Net语义分割，得到建筑物和道路的掩码（开闭操作对掩码细化），计算Hu矩进行暴力匹配，选择最好的匹配用于优化单应矩阵。

  • Marcu A., Costea D., Slusanschi E., Leordeanu M.《A multi-stage multi-task neural network for aerial scene interpretation and geolocalization》

    提出多阶段多任务(MSMT)神经网络架构，可同时执行航空图像分割和地理定位。第一阶段，语义分割网络获得道路分割；第二阶段第一分支，经纬度坐标回归；第二分支，UAV在观测图的位置估计。得到UAV位置后使用ICP优化，将道路分割和无人机附件地图道路精准对齐。

    缺点：道路分割网络的使用将解决方案限制在城市地区；经纬度回归器必须在飞行前用飞行区域的图像进行训练。

  • Goforth H., Lucey S.《GPS-denied UAV localization using pre-existing satellite imagery》 有源码

    在优化框架内提出一种结合CNN和VO的AVL方法。DEEP+ICLK，将AVL结合到VO中优化，在滑动窗口添加位姿优化，但只能减少漂移不能消除漂移。

  • Schleiss M.《Translating aerial images into street-map representations for visual self-localization of uavs》

    提出基于条件生成对抗网络(cGAN)和模板匹配的方法。作者有意避免了VO概念以获得纯AVL方法。

    第一步，将当前UAV观测从图像转换为类似于地理地图表示，即map-like image，而无需卫星图像覆盖，通过cGAN对图像分割为三类： 道路，建筑、背景。然后和参考地图匹配，模板匹配技术使用归一化平方差(SSD)方法，匹配假定rotate和scale已知可用IMU获得，因此需要根据其将模板对齐再匹配。

4.Visual odometry
4.Visual odometry

VO 是一种基于比较相机捕获的连续帧的方法，在飞行过程中累积帧之间的运动差异以获得定位估计，适合 RVL，但可以通过使用以前飞行的数据扩展构建 AVL 系统，在这种方法中，用VO收集的数据被地理参考并收集在数据库中，以便稍后重新用于定位。

  • Warren M., Greeff M., Patel B., Collier J., Schoellig A.P., Barfoot T.D.《There’s no place like home: Visual teach and repeat for emergency return of multirotor UAVs during GPS failure》

    提出一种GPS失效后可返回起始位置的方法，该系统使用云台稳定的灰度校准立体视觉系统来获取数据。…

可能的研究趋势

  1. 航空影像专用特征点方法的开发：缺乏专门的特征点检测和提取方法的研究，可进一步探索。
  2. 视觉里程计AVL：少量研究将VO应用于AVL，证明了其潜力，有待进一步研究。
  3. 深度学习 AVL：深度学习方法有局限，ALV特点使泛化性难以保证，需开展更多工作来克服挑战减少局限性。
  4. AVL公开可用数据集：包括卫星图像、真实无人机飞行数据、惯性数据。
  5. 开源代码：以实现更好的重复性。
  6. 嵌入式 GPU 实现
  7. Hyper-spectral AVL： 在高光谱定位领域做的工作很少，据我们所知，在 AVL 领域没有做。 例如，将短波红外 (SWIR) 图像配准到 RGB 卫星参考地图，因为它可以在云和雾存在的情况下进行视觉定位。
